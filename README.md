# pf2e-ja
パスファインダー2e日本語化計画

翻訳計画
１．とりあえず遊べるようにする
２．セブンドゥームオブサンドポイントを日本語で遊びやすくする
３．コンペンディウムを日本語化する
４．他のシナリオを日本語化する。

翻訳指針
カタカナを積極的に多様する。
→Achive of Nethhys等で原文を確認する事を容易にするため
→他のGMがどんな判断を下しているか、ネットでの検索を容易にするため
→公式がD&Dとの決別を図っているため、日本語化もそれなりにそれっぽくするため
→英語のゲームに日本語を当てはめる事がナンセンスだと思うため
→カタカナ用語は日本にかなり浸透しているため、カタカナの方がイメージが湧くと思うため
→慣れてきたら結局英語名で伝え合うと思うので、あまり意味をなさないと感じるため

特徴、特技、技能はそもそも漢字が被っているので、フューチャー、フィート、スキル

馴染みのあるステータスで英語→日本語の翻訳が一般的であれば日本語に訳す。attribute→能力値

スキルの名前は全てカタカナに統一する
→ロアは悩ましいが、下手に翻訳すると英語シナリオの進行で詰まる恐れがある。最低でも戦争/warのロア、などの原文を必ず載せる

フィートやクラス固有のスキル名、装備、マジックアイテム、スペルなどは全てカタカナで統一する。
→例外として中華由来や日本由来の武器は漢字にした方が良いかもと悩んでいる

説明文や、派生する単語については日本語訳にする
→具体的にはAchive of Neshysで検索用語になりづらい単語は積極的に日本語にし、原文よりも、日本語としての伝わりやすさを重視する

カタカナで表現した場合、発音が被って分かりづらい場合は、プレイ感を重視して日本語化する

状態異常は日本語化し、プレイ中にどんな状態になっているのか瞬時にイメージしやすくする
→説明文はFVTTですぐに確認できるため、検索の必要性はほぼない上に、状態異常は計算式に組み込まれる
