# pf2e-ja
パスファインダー2e日本語化計画

## 翻訳計画

1. とりあえず遊べるようにする  
2. セブン・ドゥーム・オブ・サンドポイントを日本語で遊びやすくする  
3. コンペンディウムを日本語化する  
4. 他のシナリオを日本語化する  

## 翻訳指針

### 用語表記の基本方針

- カタカナを積極的に多用する  
  - Archive of Nethys 等で原文確認を容易にするため  
  - 他の GM の判断やネット検索を容易にするため  
  - 公式が D&D との決別を図っているため、日本語化もそれに寄せるため  
  - 英語ゲームに無理に日本語を当てはめること自体がナンセンスだと考えているため  
  - カタカナ用語は日本語として既に広く浸透しているため  
  - 最終的には英語名でやり取りされることが多く、完全翻訳の意味が薄いため  

### 能力・用語の扱い

- 特徴・特技・技能は漢字が被りやすいため、以下のように表記する  
  - トレイト 
  - フィート  
  - スキル  

- 一般的に定着しているステータスは日本語訳を使用する  
  - 例：attribute → 能力値  

### スキル名について

- スキル名はすべてカタカナ表記に統一する  
  - Lore（ロア）は特に注意が必要  
  - 誤訳によるシナリオ進行停止を防ぐため、戦争/warロアのように原文併記を行う  

### クラス能力・アイテム・スペル

- フィート、クラス固有能力、装備、マジックアイテム、スペルはすべてカタカナ表記  
- 例外として、中華由来・日本由来の武器については漢字表記も検討中  

### 説明文・フレーバーテキスト

- 説明文や派生語は日本語訳を優先する  
  - Archive of Nethys で検索対象とならなさそうなものは、検索性よりも日本語としての分かりやすさを重視する  

### 読みやすさとしての例外

- カタカナ表記で発音が被り、判別しづらい場合は日本語訳を採用する    

### 状態異常（Conditions）

- 状態異常はすべて日本語化する  
  - プレイ中に状態を即座にイメージできることを重視  
  - Foundry VTT 上ですぐに詳細確認できるため、検索性は重視しない  
  - 状態異常は計算式に組み込まれるため、あまり検索される事はないと思われる
